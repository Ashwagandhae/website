    Docx Reader

    Fast docx file opener for debate

    Debate has many large documents that take forever to open, which can be detrimental when in a round. Docx reader solves this problem, by using the power of Rust to make opening documents nearly instant. Its multi-colored windows and advanced search features are sure to impress.
